# 🗄️ Northwind Satış Analizi — SQL

## 📌 Proje Hakkında
Microsoft'un klasik Northwind veritabanı kullanılarak satış analizi yapılmıştır.
Microsoft SQL Server ve SSMS kullanılarak gerçek iş soruları SQL ile cevaplanmıştır.
MySQL syntax'ı ile de uyumlu sorgular yazılmıştır.

## 🗃️ Veritabanı Hakkında
- Northwind Traders adlı hayali bir ticaret şirketinin verisi
- Müşteriler, siparişler, ürünler, çalışanlar tablolarını içerir
- 8 ürün kategorisi, 77 ürün, 91 müşteri, 830 sipariş

## 📊 Yapılan Analizler
1. En çok sipariş veren müşteriler
2. En çok gelir getiren müşteriler (TOP 10)
3. Kategori bazında satış analizi
4. Aylık satış trendi (1996-1998)
5. En çok satan ürünler (TOP 10)

## 💡 Öne Çıkan Bulgular
- QUICK-Stop (Almanya) 110,277$ ile en değerli müşteri
- Beverages kategorisi 267,868$ ile en yüksek geliri sağlıyor
- Côte de Blaye en çok gelir getiren ürün (141,396$)
- 1997 Ocak ayında 61,258$ ile en yüksek aylık satış

## 🔧 Kullanılan Teknolojiler
- Microsoft SQL Server 2017
- SQL Server Management Studio (SSMS)
- T-SQL / MySQL

## 🚀 Nasıl Çalıştırılır?
1. SQL Server ve SSMS kur
2. Northwind veritabanını yükle
3. northwind_analiz_mysql.sql dosyasını SSMS'de aç ve çalıştır
